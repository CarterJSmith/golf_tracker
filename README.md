# Overview
As my introduction to mobile application development, this project served as a deep dive into the Flutter ecosystem, allowing me to practice implementing intuitive user interfaces and interacting with data from a user.

This application is a digital golf scorecard and performance tracker designed to streamline how players record their rounds. Users can create custom scorecards for various courses and track their progress over time. This app is minimalistic by design but allows users to see only the most relivant information to them. 

The purpose of creating this specific app was to use a practical, data-driven scenario to drive my learning. A golf scorecard requires saving structured data and building interactive user interfaces. This made it an ideal candidate for mastering the fundamentals of mobile development without getting overwhelmed by complex backend infrastructure on my first attempt.
